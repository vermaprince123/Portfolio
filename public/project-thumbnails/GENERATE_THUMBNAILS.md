## Thumbnail Specifications

- **Format**: PNG or JPG
- **Size**: 800x600px (4:3 aspect ratio) or 1200x675px (16:9)
- **File naming**: Use kebab-case matching the project names above

## Generating Thumbnails

### Option 1: Screenshot from GitHub
1. Visit your repository on GitHub
2. Take a screenshot of the README or main page
3. Crop and resize to recommended dimensions
4. Save with the appropriate filename

### Option 2: Create Custom Thumbnails
You can create custom thumbnails using:
- **Figma** - Design custom project cards
- **Canva** - Use templates for project showcases
- **Screenshots** - Capture your running applications

### Option 3: Use Placeholder Generator
The portfolio automatically generates beautiful gradient placeholders with icons if thumbnails are not found. However, custom thumbnails will make your portfolio stand out more.

## Current Status

All projects currently use auto-generated gradient placeholders with project-specific icons and colors. You can replace these by adding the actual image files to this directory.
