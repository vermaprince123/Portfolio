# Project Thumbnails

Place your project thumbnail images here. 

## Recommended Specifications:
- **Format**: PNG or JPG
- **Size**: 800x600px or 16:9 aspect ratio
- **File naming**: Use descriptive names matching your project titles

## Example:
- `ishant-commerce.png`
- `ecommerce-platform.png`
- `task-manager.png`

You can update the thumbnail paths in `lib/data.ts` to match your actual image files.
